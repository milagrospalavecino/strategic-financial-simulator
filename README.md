Dynamic financial dashboard built in Power BI to simulate revenue projections using What-if parameters and DAX.
Strategic Financial Simulator (Power BI)

 Project Overview
This project features a dynamic financial dashboard developed in Power BI Desktop. It is designed to help stakeholders visualize revenue projections by adjusting price variables in real-time using "What-if" parameters.

 Key Technical Features
* Dynamic Projections: Integrated a "What-if" parameter to simulate price adjustments ranging from -50% to +50%.
* Advanced DAX: Developed the `Projected Revenue` measure to calculate real-time financial impacts.
* Data Modeling: Established a dedicated Calendar Table to ensure accurate chronological sorting and time-series analysis.
* Executive Visualization: High-impact bar charts comparing Baseline vs. Projected revenue.

 Repository Contents
- Financial Impact Analysis (WhatIf).pbix: Full Power BI Desktop file including the data model and DAX measures.
- Documentation: Detailed process documentation for technical review.
